Composes <a href="/#/PseudoBox">PseudoBox</a>.

The `Img` component provides some sane defaults when wanting to add an img to your app and
accepts all configurations that a typical img would.

```typescript jsx
<Img
  nativeWidth={150}
  nativeHeight={150}
  src="https://avatars2.githubusercontent.com/u/10436045?s=400&u=09293d4486f1fd66ee75d014821062c20b33e13e&v=4"
  alt="avatar"
/>
```
