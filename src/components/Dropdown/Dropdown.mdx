This module exports the following components:

- Dropdown
- DropdownButton
- DropdownMenu
- DropdownItem
- DropdownLink

A dropdown can define a custom button:

```typescript jsx
<Dropdown>
  <DropdownButton as={Button} variant="outline">
    Hello
  </DropdownButton>
  <DropdownMenu>
    <DropdownItem onSelect={() => alert('clicked first')}>First option</DropdownItem>
    <DropdownItem onSelect={() => alert('clicked second')}>Second option</DropdownItem>
  </DropdownMenu>
</Dropdown>
```

A dropdown can have disabled options:

```typescript jsx
<Dropdown>
  <DropdownButton as={Button}>Hello</DropdownButton>
  <DropdownMenu>
    <DropdownItem onSelect={() => alert('clicked first')}>First option</DropdownItem>
    <DropdownItem onSelect={() => alert('clicked second')} disabled>
      Second option
    </DropdownItem>
    <DropdownItem onSelect={() => alert('clicked third')}>Third option</DropdownItem>
  </DropdownMenu>
</Dropdown>
```

A dropdown can contain links as options. Use `as={Link}` if you don't want to render the default
`<a>` tag:

```typescript jsx
<Dropdown>
  <DropdownButton as={Button} varriantColor="teal">
    Hello
  </DropdownButton>
  <DropdownMenu>
    <DropdownItem onSelect={() => alert('clicked first')}>First option</DropdownItem>
    <DropdownLink href="https://google.com" target="_blank" rel="noopener noreferrer">
      I am a link
    </DropdownLink>
  </DropdownMenu>
</Dropdown>
```
