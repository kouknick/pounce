A Badge should be used to render statuses

A badge can be `solid`:

```typescript jsx
<Box bg="navyblue-500" p={3}>
  <Badge color="red-200" mb={2}>
    INFO
  </Badge>
  <Badge color="teal-300" mb={2}>
    LOW
  </Badge>
</Box>
```

A badge can be `stroke`:

```typescript jsx
<Box bg="navyblue-500" p={3}>
  <Badge color="red-200" mb={2} variant="stroke">
    INFO
  </Badge>
  <Badge color="teal-300" mb={2} variant="stroke">
    LOW
  </Badge>
</Box>
```
