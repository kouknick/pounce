A Badge should be used to render statuses

A badge can be `solid`:

```typescript jsx
<SimpleGrid spacing={3} columns={3}>
  <Badge color="red-300">INFO</Badge>
  <Badge color="teal-500">LOW</Badge>
  <Badge color="blue-400">WHATEVER</Badge>
</SimpleGrid>
```

A badge can be `outline`:

```typescript jsx
<SimpleGrid spacing={3} columns={3}>
  <Badge color="red-300" variant="outline">
    INFO
  </Badge>
  <Badge color="teal-500" variant="outline">
    LOW
  </Badge>
  <Badge color="blue-400" variant="outline">
    WHATEVER
  </Badge>
</SimpleGrid>
```
