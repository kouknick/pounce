A Badge should be used to render statuses

A badge can be `solid`:

```typescript jsx
<Box>
  <Badge color="red-200" mb={2}>
    INFO
  </Badge>
  <Badge color="teal-300" mb={2}>
    LOW
  </Badge>
</Box>
```

A badge can be `outline`:

```typescript jsx
<Box>
  <Badge color="red-200" mb={2} variant="outline">
    INFO
  </Badge>
  <Badge color="teal-300" mb={2} variant="outline">
    LOW
  </Badge>
</Box>
```
