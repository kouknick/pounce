A button can be have multiple colors and variants:

```typescript jsx
<Box backgroundColor="navyblue-600" p={5}>
  <SimpleGrid columns={3} gap={4}>
    <Button>Button</Button>
    <Button color="teal">Button</Button>
    <Button variant="outline">Button</Button>
  </SimpleGrid>
</Box>
```

A button can contain an icon:

```typescript jsx
<SimpleGrid columns={2} gap={2}>
  <Button icon="add-user">Button</Button>
  <Button icon="arrow-back" iconAlignment="right">
    Button
  </Button>
</SimpleGrid>
```

A button be marked as loading:

```typescript jsx
<SimpleGrid columns={2} gap={2}>
  <Button loading color="teal">
    Button
  </Button>
  <Button loading disabled color="teal">
    Button
  </Button>
</SimpleGrid>
```

A button can occupy the full width oof its parent:

```typescript jsx
<Button color="teal" fullWidth>
  Button
</Button>
```
