A button can be have multiple colors and variants:

```typescript jsx
<SimpleGrid columns={5} gap={4}>
  <Button>Button</Button>
  <Button variantColor="teal">Button</Button>
  <Button variant="outline">Button</Button>
  <Button variant="outline" variantColor="teal">
    Button
  </Button>
  <Button variant="outline" variantColor="navyblue">
    Button
  </Button>
</SimpleGrid>
```

A button can contain an icon:

```typescript jsx
<SimpleGrid columns={2} gap={2}>
  <Button icon="add-user">Button</Button>
  <Button icon="arrow-back" iconAlignment="right">
    Button
  </Button>
</SimpleGrid>
```

A button be marked as loading:

```typescript jsx
<SimpleGrid columns={2} gap={2}>
  <Button loading variantColor="teal">
    Button
  </Button>
  <Button loading disabled variantColor="teal">
    Button
  </Button>
</SimpleGrid>
```

A button can occupy the full width oof its parent:

```typescript jsx
<Button variantColor="teal" fullWidth>
  Button
</Button>
```
