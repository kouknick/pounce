Tabs can be really simple:

```typescript jsx
const Example = () => {
  const tabs = ['Traits', 'Event History', 'Identities'];

  return (
    <Tabs>
      <TabList>
        {tabs.map(t => (
          <Tab key={t}>{t}</Tab>
        ))}
      </TabList>
      <TabPanels>
        {tabs.map(t => (
          <TabPanel lazy unmountOnInactive key={t}>
            Panel {t}
          </TabPanel>
        ))}
      </TabPanels>
    </Tabs>
  );
};

<Example />;
```

Tabs can have disabled tabs:

```typescript jsx
const Example = () => {
  const tabs = ['Traits', 'Event History', 'Identities'];
  const [activeTab, setActiveTab] = React.useState(0);

  return (
    <Tabs onChange={setActiveTab} index={activeTab}>
      <TabList>
        {tabs.map((t, index) => (
          <Tab disabled={index === 1} key={t}>
            {t}
          </Tab>
        ))}
      </TabList>
      <TabPanels>
        {tabs.map(t => (
          <TabPanel key={t}>
            <Card p={9} mt={3}>
              Panel {t}
            </Card>
          </TabPanel>
        ))}
      </TabPanels>
    </Tabs>
  );
};

<Example />;
```

Tabs can be be lazy loaded

```typescript jsx
const Timer = () => {
  const [counter, setCounter] = React.useState(0);
  React.useEffect(() => {
    const timeout = setTimeout(() => {
      setCounter(counter + 1);
    }, 1000);
    return () => clearTimeout(timeout);
  }, [counter]);

  return <Card p={9}>{counter}</Card>;
};

const Example = () => {
  const tabs = ['Traits', 'Event History', 'Identities'];

  return (
    <Tabs>
      <TabList>
        {tabs.map(t => (
          <Tab key={t}>{t}</Tab>
        ))}
      </TabList>
      <TabPanels>
        {tabs.map(t => (
          <TabPanel lazy key={t}>
            <Timer key={t} />
          </TabPanel>
        ))}
      </TabPanels>
    </Tabs>
  );
};

<Example />;
```
