Tabs can be really simple:

```typescript jsx
const Example = () => {
  const tabs = ['Traits', 'Event History', 'Identities'];
  const [activeTab, setActiveTab] = React.useState(0);

  return (
    <Tabs onChange={setActiveTab} index={activeTab}>
      <TabList>
        {tabs.map((t, index) => (
          <Tab key={t}>{t}</Tab>
        ))}
      </TabList>
      <TabPanels>
        {tabs.map(t => (
          <TabPanel key={t}>
            <Card p={9} mt={3}>
              Panel {t}
            </Card>
          </TabPanel>
        ))}
      </TabPanels>
    </Tabs>
  );
};

<Example />;
```

Tabs can have disabled tabs:

```typescript jsx
const Example = () => {
  const tabs = ['Traits', 'Event History', 'Identities'];
  const [activeTab, setActiveTab] = React.useState(0);

  return (
    <Tabs onChange={setActiveTab} index={activeTab}>
      <TabList>
        {tabs.map((t, index) => (
          <Tab key={t} disabled={index == 1}>
            <Box p={5}>{t}</Box>
          </Tab>
        ))}
      </TabList>
      <TabPanels>
        {tabs.map(t => (
          <TabPanel key={t}>
            <Card p={9} mt={3}>
              Panel {t}
            </Card>
          </TabPanel>
        ))}
      </TabPanels>
    </Tabs>
  );
};

<Example />;
```

Tabs can be vertical

```typescript jsx
const Example = () => {
  const tabs = ['Traits', 'Event History', 'Identities'];
  const [activeTab, setActiveTab] = React.useState(0);

  return (
    <Tabs onChange={setActiveTab} index={activeTab} orientation="vertical">
      <TabList>
        {tabs.map((t, index) => (
          <Tab key={t}>
            <Box p={5}>{t}</Box>
          </Tab>
        ))}
      </TabList>
      <TabPanels as={Box} ml={3}>
        {tabs.map(t => (
          <TabPanel key={t}>
            <Card p={9} mt={3}>
              Panel {t}
            </Card>
          </TabPanel>
        ))}
      </TabPanels>
    </Tabs>
  );
};

<Example />;
```
