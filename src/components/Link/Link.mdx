Links can have a variant:

```typescript jsx
<Box>
  <Box mb={3}>
    <Link variant="prominent" href="/#/Box">
      I am a prominent link
    </Link>
  </Box>
  <Box mb={3}>
    <Link variant="neutral" href="/#/Box">
      I am a neutral link
    </Link>
  </Box>
  <Box mb={3}>
    <Link variant="discreet" href="/#/Box">
      I am a discreet link
    </Link>
  </Box>
</Box>
```

A link can be disabled:

```typescript jsx
<Link href="https://google.com" disabled>
  I am a disabled link
</Link>
```

A link can be external:

```typescript jsx
<Link href="https://google.com" external>
  I am an external link
</Link>
```

A link can implement a React-router `Link` or anything you might want:

```typescript jsx
<Link as={Heading} size="large" href="https://google.com">
  I am a link rendered as a Heading
</Link>
```
