An input can be valid:

```typescript jsx
<TextInput truncated label="Name" name="hi" placeholder="Type something..." />
```

An input can be invalid:

```typescript jsx
<TextInput label="Name" name="hi" placeholder="Type something..." value="Invalid Value" invalid />
```

An input can be disabled:

```typescript jsx
<TextInput label="Name" name="hi" disabled placeholder="Type something..." value="Disabled Value" />
```
