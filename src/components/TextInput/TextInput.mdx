An input can be valid:

```typescript jsx
<Box backgroundColor="navyblue-600" width="100%" p={6}>
  <TextInput truncated label="Name" name="hi" placeholder="Type something..." />
</Box>
```

An input can be invalid:

```typescript jsx
<Box backgroundColor="navyblue-600" width="100%" p={6}>
  <TextInput
    label="Name"
    name="hi"
    placeholder="Type something..."
    value="Invalid Value"
    error="Hello"
  />
</Box>
```

An input can be disabled:

```typescript jsx
<Box backgroundColor="navyblue-600" width="100%" p={6}>
  <TextInput
    label="Name"
    name="hi"
    disabled
    placeholder="Type something..."
    value="Disabled Value"
  />
</Box>
```
