An input can be valid:

```typescript jsx
<Box backgroundColor="navyblue-600" width="100%" p={6}>
  <NumberInput label="Minutes" placeholder="Type a number..." />
</Box>
```

An input can have `max` and `min` values:

```typescript jsx
<Box backgroundColor="navyblue-600" width="100%" p={6}>
  <NumberInput label="Minutes" placeholder="Type a number..." min={0} max={20} />
</Box>
```

An input can have a `step`:

```typescript jsx
<Box backgroundColor="navyblue-600" width="100%" p={6}>
  <NumberInput label="Minutes" placeholder="Type a number..." step={5} />
</Box>
```

An input can be invalid:

```typescript jsx
<Box backgroundColor="navyblue-600" width="100%" p={6}>
  <NumberInput label="Name" placeholder="Type a number..." value={-5} invalid />
</Box>
```

An input can be disabled:

```typescript jsx
<Box backgroundColor="navyblue-600" width="100%" p={6}>
  <NumberInput label="Name" disabled placeholder="Type a number..." value={0} />
</Box>
```
