The Modal is a controlled component. This means that you should control its
opening and closing from "outside".

When providing a `title`, the `aria-labeledby` attribute is automatically populated for you.
Depending on the content, you may opt to supply a `aria-describedby` attribute yourself

A modal can be simple:

```typescript jsx
const Example = () => {
  const [open, setOpen] = React.useState(false);

  return (
    <div>
      <Button onClick={() => setOpen(true)}>Open Modal No.1</Button>
      <Modal open={open} onClose={() => setOpen(false)}>
        <Flex justify="center">
          <Button variant="primary" onClick={() => setOpen(false)} size="small" mt={3}>
            Click me to close
          </Button>
        </Flex>
      </Modal>
    </div>
  );
};

<Example />;
```

A modal can have a title:

```typescript jsx
const Example = () => {
  const [open, setOpen] = React.useState(false);

  return (
    <div>
      <Button onClick={() => setOpen(true)}>Open Modal No.1</Button>
      <Modal open={open} onClose={() => setOpen(false)} title="Hello">
        <Flex justify="center">
          <Button variant="primary" onClick={() => setOpen(false)} size="small" mt={3}>
            Click me to close
          </Button>
        </Flex>
      </Modal>
    </div>
  );
};

<Example />;
```

A modal can have a close button:

```typescript jsx
const Example = () => {
  const [open, setOpen] = React.useState(false);

  return (
    <div>
      <Button onClick={() => setOpen(true)}>Open Modal No.1</Button>
      <Modal open={open} onClose={() => setOpen(false)} title="Hello" showCloseButton>
        Whatever
      </Modal>
    </div>
  );
};

<Example />;
```
