A Stack can be horizontal:

```typescript jsx
<Box width={300}>
  <Stack spacing={3}>
    <Box>Hi</Box>
    <Box>There</Box>
    <Box>Friend</Box>
  </Stack>
</Box>
```

A Stack can be vertical:

```typescript jsx
<Box width={300}>
  <Stack spacing={3} orientation="vertical">
    <Box>Hi</Box>
    <Box>There</Box>
    <Box>Friend</Box>
  </Stack>
</Box>
```
