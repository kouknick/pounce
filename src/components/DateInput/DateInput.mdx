As with every other form component, the DateInput is a controlled component and so it
requires for you to specify the `value` and the `onChange` handler.

Any extra prop that you specify will be given to the underlying input element of the module.

---

```typescript jsx
const Example = () => {
  const [date, setDate] = React.useState(new Date());

  return (
    <>
      <p>
        <DateInput onChange={setDate} value={date} placeholder="Enter a date" label="Start date" />
      </p>
    </>
  );
};

<Example />;
```

A DateInput also manipulates the format of the incoming date:

```typescript jsx
const Example = () => {
  const [date, setDate] = React.useState(new Date());

  return (
    <Box>
      <Box mb={4}>
        Selected date is {date.toLocaleDateString()} {date.toLocaleTimeString()}
      </Box>
      <Box>
        <DateInput
          onChange={setDate}
          withTime
          format="MM/DD/YYYY hh:mm A"
          value={date}
          placeholder="Enter a date"
          label="Start date"
        />
      </Box>
    </Box>
  );
};

<Example />;
```
